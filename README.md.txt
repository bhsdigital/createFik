# fikCalculator
Takes an invoice number and calculates the complete +71 fik number.

Pass the invoice number as the parameter, it does not matter if there is any prepended 0's or not!
This function only takes invoice numbers.
The Appended "0" which means "invoice function" should NOT be appended by the user,
only the raw invoice number

Examples:

invoice number: 21186312 
fik number    : 000002118631205

invoice number: 62147823 
fik number    : 000006214782309

invoice number: 0000012369714 
fik number    : 000001236971402
